# Typography Pairings

Pick one pairing and keep it consistent. Avoid default system stacks.

- Space Grotesk (display) + Manrope (text)
- Sora (display) + DM Sans (text)
- Instrument Serif (display) + Instrument Sans (text)
- Clash Display (display) + General Sans (text)
- Fraunces (display) + Work Sans (text)
- Source Serif 4 (display) + Source Sans 3 (text)

If licensing constraints exist, prefer open-source families from Google Fonts.
