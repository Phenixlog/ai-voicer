# App UI Checklist

- Navigation: topbar or sidebar, active states, clear hierarchy.
- Overview: dashboard with key metrics and primary actions.
- Tables and lists: sortable headers, filters, bulk actions.
- Search and filters: global search, saved filters, tags.
- Forms: grouped fields, inline help, validation feedback.
- States: loading, empty, error, success, offline.
- Details panels: side sheets or drawers for editing.
- Settings: account, billing, integrations, team.
- Onboarding: first-run checklist, guided steps.
- Notifications: toasts, inbox, audit log.
- Security cues: role badges, access hints, permissions summary.
