# Design Dimensions

Use this list to define the visual direction and avoid generic output. Pick 2 or 3 dimensions to make distinctive, then keep the rest restrained.

- Layout and grid: container widths, column count, breakpoints, whitespace rhythm.
- Typography: display vs text contrast, scale steps, line height, tracking.
- Color system: neutral base, primary, accent, success, warning, error, surface layers.
- Depth: shadow levels, borders, elevation, overlays.
- Shape: radius style, edge sharpness, card silhouette.
- Imagery: product screenshots, abstract gradients, geometric shapes, texture.
- Motion: easing, duration, delay, page-load reveals.
- Density: comfortable, default, compact spacing sets.
- Emphasis: hero scale, CTA hierarchy, callouts, badges.
- Content tone: short, punchy copy vs informative, narrative copy.
